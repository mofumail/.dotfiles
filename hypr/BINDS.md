# Keybinds Reference

SUPER = Windows/Meta key

## Core
| Bind | Action |
|------|--------|
| SUPER + Q | Open terminal (kitty) |
| SUPER + C | Close active window |
| SUPER + M | Exit Hyprland |
| SUPER + E | Open file manager (dolphin) |
| SUPER + R | Open launcher (anyrun) |
| SUPER + V | Toggle floating |
| SUPER + L | Lock screen (swaylock) |
| SUPER + N | Toggle notification center |

## Layout (Dwindle)
| Bind | Action |
|------|--------|
| SUPER + P | Toggle pseudotile |
| SUPER + J | Toggle split direction |

## Focus
| Bind | Action |
|------|--------|
| SUPER + Arrow Keys | Move focus (left/right/up/down) |

## Workspaces
| Bind | Action |
|------|--------|
| SUPER + 1-0 | Switch to workspace 1-10 |
| SUPER + SHIFT + 1-0 | Move window to workspace 1-10 |
| SUPER + Scroll | Cycle through workspaces |
| SUPER + S | Toggle scratchpad |
| SUPER + SHIFT + S | Move window to scratchpad |

## Mouse
| Bind | Action |
|------|--------|
| SUPER + LMB drag | Move window |
| SUPER + RMB drag | Resize window |

## Screenshots
| Bind | Action |
|------|--------|
| Print | Region screenshot to clipboard |
| SHIFT + Print | Full screen screenshot to clipboard |
| SUPER + Print | Region screenshot saved to ~/Pictures/ |

## Media
| Bind | Action |
|------|--------|
| Volume Up/Down keys | Volume +/- 5% |
| Mute key | Toggle mute |
| Mic Mute key | Toggle mic mute |
| Brightness Up/Down | Brightness +/- 5% |
| Play/Pause key | Toggle playback |
| Next/Prev keys | Next/previous track |
